# My translation program

If you want to install the program follow

```

git clone https://github.com/denstiny/translation.git

```

## Installation

You can install it by using that command:

```shell

sudo make install

```

And you can run:

```shell

sudo make uninstall

```

To remove it.

Do you need these dependencies:

```

python3

requests

```
